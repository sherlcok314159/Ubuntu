### 常用命令

***1.激活蓝牙***

```bash
hciconfig #确认你的蓝牙设备是否被系统识别
hciconfig hci0 up #激活蓝牙
bluetoothctl    
scan on #搜索设备
scan off #停止搜索
pair 蓝牙地址(你需要连接的设备）
```

如果以后断了，不需要重新执行上述命令，直接设置打开蓝牙，因为蓝牙**已经被激活了**

***2.Vim编辑文件***

```bash
vim 文件名
'''
q:退出
w:保存
wq:保存并退出
！：强制某动作，后面可跟上面的操作
'''
按Esc输入:，然后按需要输入命令
```

***3.关机&重启&锁屏***

```bash
shutdown -f now #立即关机
reboot #立即重启
shutdown -h 10 #10分钟后重启
shutdown -h 7:21 #7:21
Win + L #锁屏
```

***4.查找文件或者目录是否在***

```bash
e.g.
cat /home
```

***5.删除文件或者目录***

```bash
rm -f 文件名
rm -r 目录名
```

***6.移动文件***

```bash
mv 某文件 要移到的位置
```

***7.想要重复上面命令***

```bash
直接按方向键上
```

***8.清除屏幕***

```bash
clear
```

***9.切换路径***

```bash
e.g.
cd /home/tony/ML
```

***10.当前目录***

```bash
pwd
```

***11.创建目录***

```bash
mkdir 新目录名
```

***12.复制文件或者目录***

```bash
cp 某文件名 复制到的路径
```

***13.查看进程***

```bash
ps #查看进程
kill #进程号
```
